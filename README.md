# 🏫 Repositorio General - Proyectos Buenos Aires Aprende `Talento Tech!`

Este repositorio está dedicado a los proyectos y prácticas realizadas en cursos impartidos por el programa **Buenos Aires Aprende** a cargo del Equipo **Talento Tech!** . Cada carpeta representa un curso distinto, con su respectivo README interno que documenta todos los detalles técnicos y académicos de los proyectos.

---

## 📚 Cursos y proyectos alojados

### 📘 **Curso Java Back End**

La carpeta [`Adhe.Java`](./Adhe.Java/) contiene los proyectos desarrollados durante el curso **Java Back End**:

1. **Adhe.PreEntrega**: Proyecto integrador de consola, enfocado en Programación Orientada a Objetos, colecciones y diseño modular.
   - 📖 Más detalles: [`README`](./Adhe.Java/Adhe.PreEntrega/README.md).

2. **Adhe.EntregaFinal**: Incluye un **backend** desarrollado con Spring Boot y un **frontend** básico en React, formando un sistema educativo de comercio electrónico.
   - Backend: [`README`](./Adhe.Java/Adhe.EntregaFinal/techlab-backend/README.md).
   - Frontend: [`README`](./Adhe.Java/Adhe.EntregaFinal/techlab-frontend/README.md).

---

### 🌟 **Curso React JS**

La carpeta [`Adhe.React`](./Adhe.React/) contiene los ejercicios incrementales desarrollados durante el curso **React JS**:

- Proyecto incremental: Una aplicación en **React** con **TypeScript** y **Vite**, construida de manera evolutiva a través de 16 ejercicios consecutivos.
  - 📖 Más detalles: [`README`](./Adhe.React/README.md).
