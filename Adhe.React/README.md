# 💼 Repositorio General - Proyectos del Curso React JS (Buenos Aires Aprende)

Este repositorio contiene los ejercicios y prácticas desarrolladas durante el curso **React JS - Buenos Aires Aprende**, organizados como un proyecto incremental que evolucionará a lo largo de las clases.  
Cada etapa del proyecto representa uno o más ejercicios con un enfoque en el aprendizaje progresivo de React y sus conceptos fundamentales.

---

## 📁 Proyecto `React-EjerciciosIncrementales`

La carpeta [`React-EjerciciosIncrementales`](./React-EjerciciosIncrementales/) contiene el **proyecto incremental** del curso.  
Es una **aplicación basada en React**, desarrollada utilizando **TypeScript** y **Vite** como herramienta de configuración y desarrollo.  
El proyecto se organiza en ejercicios consecutivos, donde cada uno construye sobre el anterior para aplicar e integrar los conceptos esenciales de React.

### 🧩 Descripción resumida

- **Tipo:** Aplicación React para el navegador web.
- **Lenguaje:** TypeScript.
- **Herramienta de desarrollo:** Vite.
- **Propósito:** Aprender fundamentos de React, incluyendo componentes, estados, propiedades, eventos, ciclos de vida, hooks integrados, y diseño de componentes.
- **Estructura incremental:** Cada ejercicio es la continuación del anterior, construyendo una solución más robusta y completa.
- **Extras personales:** Implementaciones opcionales de herramientas adicionales como Redux (Idea futura).
